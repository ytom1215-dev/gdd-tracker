# 🌡️ 積算温度 到達日推定アプリ (Growing Degree Days Estimator)

農業・園芸における栽培管理や生育予測において、**目標とする積算温度（有効積算温度）に到達する推定日**を算出し、視覚的に分かりやすく表示する Streamlit Webアプリケーションです。

## ✨ 主な機能

* **ハイブリッド・データ方式 (API + CSV)**
  * **過去・今年の実績**: 世界的な気象API「Open-Meteo」から日平均気温を自動取得。
  * **未来の予測**: 手元の `data (1).csv` に記録された「平年値」を自動で結合。
  * これにより、APIの取得制限を回避しつつ、確実で高精度な到達日シミュレーションを実現しています。
* **複数地点のワンタッチ切り替え**
  * 長島町、根占町、鹿屋市、南さつま市、金峰町大野 の5地点を標準搭載。選択した地点の座標に合わせてデータを自動再取得します。
* **過去年次の振り返りシミュレーション**
  * 2015年から現在まで、対象年を自由に変更可能。過去の気象実績ベースでの検証が行えます。
* **有効積算温度（Base Temperature）への対応**
  * 作物の生育が始まる「基準温度」を設定可能。基準温度を下回る日を「0」として計算する、より実践的な生育予測（GDD: Growing Degree Days）に対応しています。
* **インタラクティブなグラフ＆データ出力**
  * `Plotly` を用いた高機能グラフにより、特定の日付の数値をマウスオーバーで詳細確認可能。
  * 計算された日々のデータ（実績・平年値・積算温度）は、ワンクリックでCSVとしてダウンロードできます。

## 📁 必要なファイル構成

本アプリを動かすには、同じディレクトリに以下のファイルが必要です。

* `app.py`: アプリケーションのメインプログラム
* `data (1).csv`: 平年値データ（「年月日」と「平年値平均気温(℃)」の列が含まれていること）
* `requirements.txt`: 必要なPythonライブラリの一覧
